# Project Helix

Cross-browser browser extension built with WXT and TypeScript.

Current MVP targets Claude.ai.

Primary features:

1. Token Analytics
2. Session Refresh
3. Chat Transfer to ChatGPT

Architecture principles:

- Modular
- Strong typing
- Reusable utilities
- Production quality
- No unnecessary dependencies

Current status:

- Sidebar completed
- Conversation parser completed
- Working on Token Analytics
